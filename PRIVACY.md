# Privacy Policy – SipTime

SipTime is a browser extension designed to provide hydration reminders. Your privacy is important, and this extension is built to work entirely locally.

## Data Collection

SipTime does not collect, store, transmit, or share any personal data.

The extension does not collect:
- Personal identifiers (name, email address, etc.)
- Health data
- Financial or payment information
- Location data
- Browsing history
- Website content
- User activity or analytics

## Data Storage

All user preferences, such as reminder schedules, quiet hours, and snooze duration, are stored locally using the browser’s storage APIs. This data remains on the user’s device and is not transmitted to any external servers.

## Third-Party Access

SipTime does not sell, transfer, or share user data with third parties.

## Remote Code

SipTime does not load or execute any remote JavaScript or WebAssembly code. All functionality is included within the extension package.

## Changes to This Policy

If the functionality of SipTime changes in a way that affects user privacy, this policy will be updated accordingly.
